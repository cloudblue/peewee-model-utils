# Peewee Model Utils


## Run tests

`peewee-model-utils` uses [poetry](https://python-poetry.org/) for dependencies management and packaging.

To run the `peewee-model-utils` tests suite run:

```
$ pip install poetry
$ poetry install
$ poetry run pytest
```
